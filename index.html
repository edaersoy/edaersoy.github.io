YOLO Object Detection — Public Demo

In progress • Jan 2026

A personal project where I apply what I’m learning in Team DIANA to build an end-to-end object detection workflow with public, reproducible code (without sharing team code/data).

What I’m doing

  • Collecting images and annotating bounding boxes
  • Preparing dataset (structure, splits, sanity checks)
  • Training YOLO models and iterating on results
  • Evaluating performance (mAP, precision/recall) + error analysis
  • Running inference demo on video/webcam

Key learnings

  • Annotation consistency often beats “more data”
  • Small controlled changes + evaluation = faster progress
  • Good experiment notes make results repeatable
